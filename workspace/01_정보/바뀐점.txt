#조선 프로빈스 확장 버젼 데모
map file
1.프로빈스 추가 및 변경
	- 프로빈스 지도 변경(조선지역 및 대마도)
	- 프로빈스 이름 변경(변경사항 파일 첨부)
	- 프로빈스 추가(변경사항 파일 첨부)
	- 프로빈스 지형 변경(변경사항 파일 첨부)
	- 조선팔도 추가 및 프로빈스 추가(변경사항 파일첨부)
	- 프로빈스 기후 설정
	- 프로빈스 포지션 설정
	
	
	
2.지형 변경
	- 지형 지도 변경(강화도 추가)
	- 내해 설정(강화도-파주 구간)
	- 고도 지도 수정(강화도 구간)

history file
1. 프로빈스 정보 설정
	- 프로빈스 특산품 설정(변경사항 파일첨부)
	- 프로빈스 세금 설정

common file
1. 트레이드노드 설정

localisation file
